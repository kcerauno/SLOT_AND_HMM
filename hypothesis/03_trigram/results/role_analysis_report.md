# Trigram HMM 役割曖昧性解消 分析レポート

**生成日時**: 2026-03-08 23:15:30
**分析対象**: Trigram HMM vs Bigram HMM の Focus State 役割分離

---

## 研究仮説

Bigram HMM では Focus State が B-end (基末尾) と B-start (基先頭) の
両役割を混在して担っていた（役割曖昧性）。
Trigram HMM は 2 ステップ前の文脈 s_{t-2} を参照できるため、
この曖昧性を局所文脈で区別できるはず。

**判定基準**: B-end と B-start で s_{t-2} の分布が有意に異なるか (Fisher p < 0.05)
