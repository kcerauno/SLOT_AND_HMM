## 1. S4 → ? 実証遷移（B-end 直後の状態分布）

### 数値結果（k=7）

「B-end 位置の状態 = S4」のとき、直後の B-start 位置に現れる状態の分布（n=6,722）：

| 遷移先 | 実証確率 | 理論確率 (trans) | 差分 |
|--------|---------|----------------|------|
| S0     | 0.205   | 0.246           | −0.041 |
| S1     | 0.183   | 0.225           | −0.042 |
| S2     | 0.102   | 0.108           | −0.006 |
| S3 [Phantom] | 0.000 | 0.000        | 0.000 |
| **S4 [自己遷移]** | **0.510** | **0.421** | **+0.089** |
| S5     | 0.000   | 0.000           | 0.000 |
| S6     | 0.000   | 0.000           | 0.000 |

### 予想外の発見：自己遷移が理論値を上回る

`interpretation_notes.md` Section 5.3 では「S4→S4 自己遷移が低いため B-start が抑制される」と説明した。
しかし実証値（0.510）は理論値（0.421）を**上回った**。

**メカニズムの修正**：

単純な「自己遷移が低いから抑制される」という説明は成立しない。
より正確には、複合語境界という文脈が **S4→S4 の自己遷移を促進している**。

```
V8 境界互換文字 {l, r, o, y, s} で終わる基
  → 次の基の先頭もスロット0 {l, r, o, y, s, v} から始まる（V8制約の対称性）
  → B-end でも B-start でも同じ文字種 → HMM が S4→S4 として捉える
```

よって B-end (54%) > B-start (28%) という差は、自己遷移の抑制ではなく
**「B-end のほうが S4 適合文字の濃度が高い」**という放出確率の差によって生じている可能性が高い。

### k=8 S5 の遷移（参考）

S5 → ? 実証分布（n=2,435）：

| 遷移先 | 実証確率 | 理論確率 |
|--------|---------|--------|
| S1     | 0.479   | 0.598  |
| S3     | 0.230   | 0.212  |
| S6     | 0.291   | 0.185  |
| S5（自己） | 0.000 | 0.000 |

S5 は理論・実証ともに自己遷移ゼロ。S5 後は S1・S6・S3 に分散する。
k=8 の境界では S5 がいったん出現すると、次の B-start では必ず別の状態に遷移する。

## 2. 基数別パターン（k=7 S4）

### 数値結果

| 基数 | 語数 | B-end 率 | B-start 率 | B-end > B-start? |
|------|------|---------|-----------|----------------|
| 2基  | 5,434 | **76.3%** | 40.6% | ✓ |
| 3基  | 918   | **49.2%** | 28.8% | ✓ |
| 4基  | 1,706 | **32.7%** | 14.9% | ✓ |

B-end > B-start は全基数で成立。基数が増えるにつれて両率が単調減少する。

Fisher 検定（B-end 率の基数間比較）：

| 比較 | p値 | オッズ比 |
|------|-----|---------|
| 2基 vs 3基 | 3.5×10⁻¹⁰⁰ | 3.32 |
| 2基 vs 4基 | ≈0 | 6.62 |
| 3基 vs 4基 | 2.9×10⁻³⁵ | 1.99 |

全比較で有意差あり。基数による単調減少は統計的に確認された。

### 境界インデックス別（3基複合語）

| 境界 | B-end 率 | B-start 率 |
|------|---------|-----------|
| 境界1（base1/base2 間） | **66.6%** | 31.8% |
| 境界2（base2/base3 間） | **31.8%** | 25.7% |

境界1 >> 境界2 という非対称性が確認された。

### 境界インデックス別（4基複合語）

| 境界 | B-end 率 | B-start 率 |
|------|---------|-----------|
| 境界1（base1/base2 間） | **63.0%** | 29.6% |
| 境界2（base2/base3 間） | **29.0%** | 9.8% |
| 境界3（base3/base4 間） | **6.2%** | 5.3% |

語頭から遠ざかるにつれて S4 出現率が急落する（63% → 29% → 6%）。

### 解釈：S4 は「語頭からの距離」に敏感

この単調減少パターンは S4 が**語頭特化状態**であるという先行解釈（S-head=77%）と完全に一貫している。

```
S-head（語頭, n=4,697）:  77.3%  ← 語全体の開始点（語頭距離=0）
境界1 B-end（3基, n=918）: 66.6%  ← 語頭から最も近い境界
境界1 B-end（4基, n=1706）: 63.0%  ← 同様
境界2 B-end（3基, n=918）: 31.8%  ← 中間境界
境界3 B-end（4基, n=1706）:  6.2%  ← 語末に最も近い境界
S-mid（語中央, n=4,697）:   3.8%  ← 語の中央（境界なし）
```

語頭（77%）→ 第1境界（65%）→ 第2境界（30%）→ 第3境界（6%）→ 語中（4%）という
**連続的な勾配**が存在する。S4 は「語の開始点に近いほど強く活性化する状態」であり、
複合語の各基境界はその勾配上に位置している。

## 4. 先行解釈の更新

### 更新点

| 項目 | 旧解釈 | 更新後解釈 |
|------|--------|----------|
| B-end > B-start の原因 | S4 自己遷移が低いため B-start で抑制 | 自己遷移は実証で促進される（0.51）。B-end の方が S4 適合文字の濃度が高いことが主因 |
| 基数不変性 | 複合語構造が安定なら基数によらず一定のはず | 実際は基数増加に従い単調減少。S4 は語頭距離依存の勾配を持つ |

### 維持される解釈

- **B-end vs S-mid の純粋構造シグナル**（54% vs 4%, odds=29.9）は変わらない
- **交絡仮説の棄却**（Fisher①: B-start << S-head）は変わらない
- **複合語仮説の支持**（V8 × HMM の独立一致）は変わらない

## 5. 総合的含意

### S4（k=7）の機能モデルの精緻化

```
語頭からの距離 d に対して:
  S4 出現率 ≈ f(d), f は単調減少関数

  d=0 (語頭):     77%
  d=1 (第1境界):  65%  ← 複合語の最初の基の終端
  d=2 (第2境界):  30%
  d=3 (第3境界):   6%
  d=∞ (語中央):   4%
```

S4 は「語・基の開始点に近い位置での放出確率が高い V8 境界互換文字に特化した状態」であり、
かつその活性化は語頭からの距離に従って減衰する勾配構造を持つ。

### 複合語仮説への含意

基数増加に伴う単調減少は、複合語仮説を**弱める**方向にも解釈できる：
→「S4 が複合語の"すべての境界"を均等にマーキングする」とは言えない。

しかし同時に、この単調減少自体が**複合語の内部構造**を反映している：
→ 語頭から見た「境界の深さ」という次元で HMM が複合語を解析している可能性。
