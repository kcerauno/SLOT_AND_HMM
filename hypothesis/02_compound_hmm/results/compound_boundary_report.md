## 1. 検証の目的

ヴォイニッチ手稿の単語分析に用いた2つの独立手法、**V8スロット文法**と**HMM（隠れマルコフモデル）**の結果を相互検証する。

### 1.1 前提となる先行分析の知見

| 手法 | 知見 |
|---|---|
| V8スロット文法 | 全ユニーク単語 8,060語の大多数が「複数のベース（2〜4基）の連結」として解析できる |
| HMM (k=7, k=8) | 「Phantom State」（k=7: S3、k=8: S4）と呼ぶ縮退状態が出現。全単語の占有率が0%（`slot_hmm_state_analysis.py` で確認済み） |

### 1.2 検証仮説

> **Phantom State は BOS/EOS 位置専用のハブではなく、V8文法が独立に発見したベース間の境界位置に特化した遷移ハブとして機能している。**

もしPhantom StateがV8複合境界位置に統計的に集中するなら、2つの独立手法が同じ「複合語構造」を発見したことになり、ヴォイニッチ語の複合語構造の存在に対する強力な相互検証証拠となる。

## 2. 分析手法

### 2.1 V8複合語と境界位置の特定

`analyze_slot_grammar_v8.py` の `is_v8` 関数と同一ロジック (`find_v8_splits_first`) で、全ユニーク単語に対して2基以上の分割を試みた。

```
分割の探索順: 最初のベースが最短になる順（is_v8 の4重ループ構造と同一）
境界位置の定義:
  end-of-base  : 各ベースの最終文字位置 (= 累積長 - 1)
  start-of-base: 次ベースの先頭文字位置 (= 累積長)
  boundary_both: 上記の和集合
```

### 2.2 Viterbi デコード

キャッシュ済みHMMモデル（k=7, k=8）を用いて各複合語をViterbiデコードし、各文字位置に割り当てられた状態を取得。BOS/EOSは除外。

### 2.3 統計検定

- **カイ二乗検定**: 境界位置 vs 非境界位置での全状態の分布比較
- **Fisher正確検定**: Phantom Stateに対する境界集中の片側検定
- **シャッフル検定（Null分布）**: 境界ラベルをランダム化した1000回試行でenrichment ratioの帰無分布を生成

## 4. 解釈

### 4.1 Phantom State 仮説について

**仮説は検証不能であった。** Phantom StateはViterbiアルゴリズムによって一切選択されない真の縮退状態である。

縮退の原因：Baum-Welch（EM）学習の過程で、この状態が訓練データの文字に対して有意な放出確率を獲得できなかった。Viterbiは最大確率パスを選ぶため、全文字に対して他の状態が常に優位となり、Phantom Stateが選ばれることはない。

### 4.2 S4/S0の境界集中について：有意だが解釈に注意が必要

S4（k=7）・S0（k=8）の境界集中は統計的に極めて有意であるが、**この結果にはシンプルな説明がある**：

```
V8の is_base() は parse_greedy() を使う
↓
parse_greedy() はスロット0 (l,r,o,y,s,v) や スロット1 (q,s,d,...) を先頭から試みる
↓
新ベースの先頭文字は必然的に o, q, l, r, y などになりやすい
↓
HMMのS4/S0は独立に o, q, l のクラスターを学習済み
↓
∴ 境界位置にS4/S0が集中するのは、スロット文法の文字制約から導かれる自明な帰結である可能性がある
```

この交絡要因を排除せずには、「V8とHMMが複合語構造を独立発見した」とは断言できない。

### 4.3 対照的な発見：境界を「回避」する状態

境界に集中する状態と同様に、**境界を強く回避する状態**も確認された：

| 状態（k=7） | 差分 | 解釈 |
|---|---|---|
| S2 (A-frame) | −19.1% | 語頭・語末フレームを担う → ベース内部に収まる |
| S5 (I-run) | −18.1% | i・流音連続 → ベース末尾の典型的な接尾辞パターン |
| S0 (E-cluster) | −13.5% | e連続 → ベース内部・語幹的な位置に集中 |

これらの状態が境界を回避することは、各状態がベース内部の特定の機能的役割を担っているというHMMの先行知見と一致する。

## 5. 検証の限界と結論

### 5.1 仮説の支持・棄却判定

| 検証項目 | 判定 | 根拠 |
|---|---|---|
| Phantom State = 境界遷移ハブ仮説 | **検証不能** | Phantom StateがViterbiパスに出現しない |
| V8境界とHMM状態分布の相関 | **確認（ただし解釈困難）** | χ²検定p≈0、S4/S0が境界に集中 |
| V8とHMMによる複合語構造の独立発見 | **判断保留** | 交絡要因（スロット文法の文字制約）を排除できていない |

### 5.2 現時点での結論

> V8複合語境界とHMM状態分布の間に**統計的相関は存在する**が、その相関が「複合語構造の証拠」なのか「スロット文法の文字制約から自明に導かれる帰結」なのかは、本分析単独では判別できない。

循環論法を避けた検証のために、以下を推奨する：

```python
# → ランダム境界でも同程度S4が集中するなら自明な結果
# → V8境界のみで集中するなら複合語構造の証拠
for word in compound_words:
    random_boundary = random.randint(1, len(word) - 1)
    # 同じ分析を適用
```

### 6.2 非複合語（単独ベース語）との比較（優先度：高）

単独ベース語には「境界」が定義されないが、単独ベース語の中央位置とV8複合語の境界位置でS4出現率を比較する。

### 6.3 放出確率の直接確認（優先度：中）

S4の放出分布（どの文字に高い確率を持つか）とV8境界に出現する文字の分布を直接比較する。両者が一致するなら交絡要因による偽陽性と確定できる。

### 6.4 異なるk値での再現性（優先度：中）

k=9, 10 でモデルを再訓練し、同様のO/Q系状態の境界集中が再現するか確認する。
