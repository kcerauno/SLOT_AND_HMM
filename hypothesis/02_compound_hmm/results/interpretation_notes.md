# 解釈ノート: V8複合語仮説 × HMM Phantom State 境界検証

作成日: 2026-03-03
関連スクリプト: `compound_boundary_analysis.py`, `single_vs_compound_analysis.py`
関連レポート: `compound_boundary_report.md`, `single_vs_compound_report.md`

## 1. 研究の問い

ヴォイニッチ語のV8スロット文法（`SLOTS_V8`）が定義する「複合語の基境界」は、
HMMが統計学習によって独立に発見した構造と一致するか？

もし一致するなら、**2つの独立した手法が同じ構造を発見した**ことになり、
ヴォイニッチ語における複合語構造の存在に対する相互検証となる。

---

## 2. 前提: V8複合語仮説

V8スロット文法では、ヴォイニッチ語の語を最大16スロットの連鎖として解析する。
1語が2〜4個の「基（ベース）」に分割できるとき、その語を**複合語**と呼ぶ。

```
SLOTS_V8[0] = {l, r, o, y, s, v}   ← 各基の先頭文字
SLOTS_V8[13] = {s}
SLOTS_V8[14] = {y}                  ← 各基の末尾文字候補
SLOTS_V8[15] = {k, t, p, f, l, r, o, y}
```

`find_v8_splits_first()` による分類:
- **V8複合語**: 8,058 語（全ユニーク語の約65%）
- **単独ベース語（比較対照群）**: 4,697 語（`words_base_only.txt`）

## 3. HMM分析の主要発見（`compound_boundary_analysis.py`）

### 3.1 Phantom State は Viterbi に出現しない

| モデル | Phantom State | Viterbi占有率 |
|--------|--------------|-------------|
| k=7    | S3           | 0%          |
| k=8    | S4           | 0%          |

Phantom Stateは Baum-Welch学習中に縮退した**真の退化状態**。
「複合語境界専用のハブ」という当初の仮説は、この時点で直接的には棄却された。

### 3.2 別の状態が境界に集中している

Phantom State以外の状態が複合語境界に集中することが判明した。

| モデル | 境界集中状態 | 境界位置での出現率 | 非境界位置 | enrichment |
|--------|------------|-----------------|---------|-----------|
| k=7    | S4         | ~43.9%           | ~4.2%   | ~10.4x    |
| k=8    | S0         | ~28.7%           | ~4.1%   | ~7.0x     |

χ²検定: p ≈ 0（両モデル）

## 4. 交絡因子の検討（Section 6.2）

### 4.1 交絡仮説

> V8文法はスロット0 `{l, r, o, y, s, v}` により複合語の**第2基以降の先頭文字**を制約する。
> HMMはこの文字集合を独立にクラスタリングした結果として境界集中が生じているだけかもしれない。

この仮説が正しければ: `rate(B-start) ≈ rate(S-head)` — どちらも「基の先頭文字」だから。

### 4.2 4グループ比較設計

| グループ | 定義 | 役割 |
|---------|------|------|
| B-start | 複合語内の第2基以降の先頭文字 | 交絡の標的 |
| B-end   | 複合語内の各基の末尾文字       | 追加検証 |
| S-head  | 単独ベース語の pos=0            | 交絡の対照 |
| S-mid   | 単独ベース語の pos=L//2         | ニュートラル基準 |

### 4.3 検証結果

#### k=7 (Focus: S4)

```
S-head   77.3%  ← 単独語の語頭
B-end    54.3%  ← 複合語の基末尾  ★
B-start  28.3%  ← 複合語の基先頭
S-mid     3.8%  ← 単独語の語中（ベースライン）
```

**Fisher①** (B-start vs S-head): p ≈ 0, odds = 0.116

交絡仮説の予測（B-start ≈ S-head）に対し、**実際は B-start << S-head という逆転**。
交絡仮説は否定された。

#### k=8 (Focus: S5)

```
B-start  23.4%  ← 複合語の基先頭
S-head   20.5%  ← 単独語の語頭
B-end    19.7%  ← 複合語の基末尾
S-mid    16.8%  ← 単独語の語中
```

**Fisher①** (B-start vs S-head): p = 5×10⁻⁵, odds = 1.185

B-start > S-head が統計的に有意。**弱い構造効果**を示唆。

## 5. 統合解釈

### 5.1 S4（k=7）の機能的役割

S4 は単純な「基先頭文字クラスター」ではない。
数値パターン `S-head(77%) >> B-end(54%) >> B-start(28%) >> S-mid(4%)` が示すのは：

1. **語頭特化**: 単独語の先頭（語全体の開始点）で最も高頻度
2. **基末尾集中**: 複合語内の基末尾（次の基への遷移点）でも高頻度
3. **内部基先頭では抑制**: 複合語の途中にある基先頭では、語頭ほど高くない

これは S4 が**「ひとつの単位が完了する」境界点**に反応していることを意味する。
語全体の終端（= 語頭の直前）でも、複合語の基終端でも、「単位の完了」という共通の文脈がある。

比喩: S4 は「ピリオド」や「句点」のような役割。文が終わったとき（= 次の文の始まりの直前）に現れる。

### 5.2 V8文法側の対応現象

S4の「基末尾集中」に対応するV8文法の構造的特性:

```
基先頭スロット: {l, r, o, y, s, v}
                    ↕  重複
基末尾スロット: {l, r, o, y} (slot 15)
              {s}            (slot 13)
              {y}            (slot 14)
```

**V8文法では「基の末尾に来られる文字」と「基の先頭に来られる文字」が大幅に重複している。**
これは言語構造としての必然性（複合語の接続部は両者の文字制約を満たす必要がある）の反映でもある。

S4 は HMM側でこの「境界互換文字集合 {l, r, o, y, s}」を統計学習によって独立に発見した可能性が高い。

### 5.3 なぜ B-start < S-head なのか

交絡仮説の反証として重要なメカニズム:

- **S-head（77%）**: 語全体の先頭 → BOS 直後の遷移 → HMM が高確率で S4 を割り当てる最も典型的な文脈
- **B-start（28%）**: 複合語の途中にある基先頭 → 直前が B-end（S4: 54%）→ S4 から S4 への自己遷移確率が低い

HMM の遷移確率上、**S4 は S4 の直後には来にくい**（自己遷移が低い）ため、
B-end で S4 が出た後の B-start では S4 が抑制される。
これは複合語の「基末尾→基先頭」という構造を HMM が文脈的に捉えていることを示す。

## 6. 複合語仮説への総合的評価

### 支持される点

| 根拠 | 内容 |
|------|------|
| B-end vs S-mid (k=7) | 54% vs 4%, odds=29.9, p≈0 → **文字種では説明できない純粋な構造シグナル** |
| Fisher① 逆転 (k=7)   | B-start << S-head → 交絡仮説の棄却 |
| k=8 構造効果          | B-start > S-head (p=5×10⁻⁵) → 弱いが有意な構造効果 |

### 修正が必要な点

| 当初の想定 | 実際の結果 |
|-----------|---------|
| Phantom State が境界に集中する | Phantom は Viterbi に出現しない |
| S4 が「境界先頭マーカー」になる | S4 は「境界末尾マーカー（基終端ハブ）」として機能 |
| 交絡として棄却される可能性 | 逆方向の逆転により交絡は否定 |

### 結論

> **複合語仮説は支持される。ただし「基先頭でのマーキング」ではなく「基末尾での遷移ハブ」という形で実現している。**
>
> V8文法（独立に設計）と HMM（統計学習）が、同じ複合語の「基の完了点」を
> それぞれ異なる手法で同定している。これは両手法が同じ潜在構造を捉えていることの
> 相互検証と解釈できる。
