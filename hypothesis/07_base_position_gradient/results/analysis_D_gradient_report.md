- 正規化位置: `pos = i / max(len(base)-1, 1)`（0=語頭, 1=語末）

### 2. 事前仮説の照合（提案 A との統合）

提案 A の放射分析から「c/f 系状態は head 集中、y/h 系状態は tail 集中」という事前仮説を立てたが、**Trigram k=7 のみ完全に確認、他のモデルでは Focus State の方向が逆転した**。

#### Trigram k=7：仮説が完全成立

- B-start Focus **S2**（c=58.6% 放射）: slope = **−0.030 *** → head 集中 ✓**
  - 'c' は ch/cth/ckh 等複合子音クラスタの先頭文字 → 語頭に現れやすい → 予測通り
- B-end Focus **S5**（y=51.5% 放射）: slope = **+0.252 *** → tail 集中 ✓**
  - 'y' は Voynich 語の語末文字として典型的 → 基末尾に集中 → 予測通り

#### Bigram k=7, k=8 / Trigram k=8：B-start Focus が tail 集中（逆転）

- Bigram k=7 B-start Focus **S2**（a/y/o 放射）: slope = **+0.174 *** → tail 集中 ✗**
- Bigram k=8 B-start Focus **S1**（a/y/o 放射）: slope = **+0.180 *** → tail 集中 ✗**
- Trigram k=8 B-start Focus **S5**（y=64.0% 放射）: slope = **+0.242 *** → tail 集中 ✗**

**解釈**：これら 3 モデルの B-start Focus State は vowel 系（a/y/o）を高確率で放射する。
Voynich 語において母音的文字は語末に集中する傾向があるため、
これらの状態が「複合語境界の先頭」（= B-start）に特化しつつも、
全基内での位置分布では tail 集中を示すことは放射特性から説明可能。

つまり：**「B-start 特化」と「基内 tail 集中」は矛盾しない。**
母音的文字が各基の末尾に現れやすく、かつ次の基の先頭にも現れやすい（境界位置が母音的）。
